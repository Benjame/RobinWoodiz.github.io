优秀程序员地图

技能树

程序设计 + 软件开发

程序设计

  • 掌握常用的数据结构和算法（例如链表，栈，堆，队列，排序和散列）；
  • 理解计算机科学的核心概念（例如计算机系统结构、操作系统、编译原理和计算机网络）；
  • 熟悉至少两门以上编程语言（例如C++，Java，C#和Python）；

软件开发

  • 具备良好的编程实践，能够编写可测试（Testable），可扩展（Extensible），可维护（Maintainable）的代码；
  • 把握客户需求，按时交付客户所需要的软件产品；
  • 理解现代软件开发过程中的核心概念（例如面向对象程序设计，测试驱动开发，持续集成，和持续交付等等）。

从这两个方面着手，在程序设计方面我需要加强算法设计，计算机科学核心概念，熟悉语言这三方面的练习。

在软件开发方面，需要一方面多写代码，写大工程量的程序去体会感悟软件工程的思想，平时多看软件开发方面的书籍。

经典书籍反复读

  • 算法设计

《算法（第四版）》/《算法（C++语言描述）》、《算法导论》、《数据结构》、《编程珠玑》、《编程珠玑（续）》

  • 计算机科学核心概念

这个的核心目前本科就是『计算机组成原理』、『操作系统原理』、『计算机网络』、『数据结构』这四门专业408课程。

  • 重要书籍：《深入理解计算机系统》、《计算机组成<软硬件接口>》、《计算机网络》、《现代操作系统》、《操作系统概念》；

  • 熟悉语言

熟悉语言这方面，主攻 C++、Java，语法层次要提升到相当熟悉水平（即不翻书对语法细节能了如指掌），业余学习 C# 、Python，Golang了解爱好。

  • 重要书籍：
    • C++ ：《 C++ Primer（第5版）》、《 Effective C++》、《 More Effective C++》、《 C++标准库（C++11）》、《深度探索C++对象模型》；
    • Java：《Java核心技术卷（1）》、《Java核心技术卷（2）》、《Java编程思想》、《Effective Java》；
  • 启发书籍：
    • 程序设计原理：《程序设计方法》、《计算机程序的构造和解释》
    • 语言层次理解：《程序设计语言—实践之路》

  • 实战开发：

  • Qt 与 C++11开发应用程序，熟悉C++ 11 与 Qt库。
    • 运动提醒器
    • 网络聊天器
    • 汽车虚拟仪表
    • 音乐播放器
  • Git 持续开发
  • Linux管理与Shell编程
  • IntelliJ IDEA 配置与使用

  • 软件工程

  • 编程实践：《程序设计实践》、《 C++高质量程序设计》
  • 面向对象程序设计：《 Head First 设计模式 》、《设计模式》
  • 重构：《重构：改善既有代码的设计》
  • 专业开发：《程序员修炼之道—从小工到专家》
  • 持续集成（Continuous integration，简称CI）：什么是持续集成：相关博客；

  • 编译理论

本科阶段对编译理论就不做深究了，了解即可：可以看一本《编程语言实现模式》或者《自制编程语言》，侧重前端（Parsing Techniques）。
